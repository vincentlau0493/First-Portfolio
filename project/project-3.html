Introduction

Project Brief

This is an interface design for application on mobile. Work Phase is an application that user can record all their works and efforts when pursuing a goal. On each phase, user can score their work and the score would be add to previous score. When the percentage of certain objective becomes 100%, it means the objective has been completed and user can focus on other goals.

Idea

The idea occurs to me when I am learning ios development. It is such a frustrated stuff that I always fotgot what I have learnt yesterday. In this case, I had to waste a lot of time to review previous tutorial. Besides, tedious learning everyday lessens my passion to continue more or less. Therefore, I want to design an application which can overview what I have done or learnt before, as well as urges me to continue my study plan.

Technologies

photoshop, interface design, prototype design

Detail

Overview

Overview is the home page that displays all info of user's objectives. The cell for each objective includes date of start and progress of goal. Also, user can add a new objective when clicking the last cell.

Sidebar

Menu for user is slided from left side. It would be highlighted when clicking.

Add diary

This function allows user record his or her effort or work by saving photo, date, location and diary. Before submitting the work diary, user should select the objective which the diary is for. What's more, user should score his effort, which would be added to whole progress.

Analytics

There are two styles to overview user's diaries for each objective. It is pretty easy for user to recall and analyze their works and efforts.
